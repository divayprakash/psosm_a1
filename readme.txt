Prerequisites for running ‘twitter.py’ script -
- Tweepy
- Plotly for Python
- NLTK
- Config file ‘config.py’ containing the following 3 lines -
  - twitter = {'API_KEY': '', 'API_SECRET': ''}
  - plotly = {'API_KEY': ''}
  - facebook = {'client_id': '', 'client_secret': ''}

Prerequisites for running ‘facebook_*’ scripts -
- Facebook SDK for Python
- Plotly for Python
- Config file same as defined for Twitter script above

Prerequisites for running ‘bonus.py’ script -
- Tweepy
- Config file same as defined above
